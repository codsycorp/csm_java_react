## 📦 SOURCE CODE FILES

### New Java Services (3 files)

#### 1️⃣ AiIntentClassifierService.java
**Location**: `backend/src/main/java/net/phanmemmottrieu/service/`

**Purpose**: Fast intent classification (off-topic vs code vs menu)

**Key Methods**:
- `classify()` - Main classification method
- `classifyWithHeuristics()` - Fast pattern matching
- `classifyWithLlm()` - Optional LLM-based (slower but more accurate)
- `clearCache()` - Clear classification cache

**Features**:
- Pattern-based classification (<50ms)
- Optional LLM classification (300-500ms)
- 5-min cache with LRU eviction
- Confidence scoring

**Config**: `ai.intent.classifier.*`

---

#### 2️⃣ AiExecutionPlannerService.java
**Location**: `backend/src/main/java/net/phanmemmottrieu/service/`

**Purpose**: Generate & optimize execution plans

**Key Methods**:
- `generatePlan()` - Main planning method
- `parseStepsFromContext()` - Extract step descriptions
- `dedupAdjacentSteps()` - Merge overlapping steps
- `validatePlan()` - Verify plan correctness

**Features**:
- Smart step parsing
- Scope boundary assignment (line ranges)
- Adjacent step deduplication
- Execution time estimation

**Config**: `ai.execution.plan.*`

---

#### 3️⃣ AiScopedContextIngestionService.java
**Location**: `backend/src/main/java/net/phanmemmottrieu/service/`

**Purpose**: Smart scoped context ingestion to Lucene

**Key Methods**:
- `analyzeScopesFromAttachments()` - Determine what to index
- `ingestCode()` - Index code with scope filtering
- `ingestMenu()` - Index menu with scope filtering
- `waitForIngestion()` - Wait for async completion

**Features**:
- Scope mask analysis from attachments
- Smart content chunking (2200 chars default)
- Async + sync ingestion modes
- Auto-pruning (keep 48 sources)

**Config**: `ai.context.ingestion.*`

**Scope Masks**:
- `SCOPE_CODE` (0x01) - Current code file
- `SCOPE_MENU` (0x02) - Menu structure
- `SCOPE_CONFIG` (0x04) - Configuration
- `SCOPE_EXTERNAL` (0x08) - Attachment data
